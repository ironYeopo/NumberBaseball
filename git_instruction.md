일반 폴더를 깃 폴더로
git init

바꾼것을 stage 할때
git add filename1 filename2 ...
OR
git add * (전부다)

commit 할때
git commit -m "메시지"

리모트 서버로 push 할때
git push origin master
master 브랜치를 origin으로 푸쉬하겠다 

현 상황을 보고싶다
git status

리모트 서버를 로컬로 불러올떄
git pull

브랜치를 만들때
git checkout -b 브랜치이름 (그 브랜치로 가짐)

브랜치로 이동
git checkout 브랜치이름
